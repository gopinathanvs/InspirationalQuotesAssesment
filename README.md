1. **Open the Solution:**
   Navigate to `InspirationalQuotesAssessment\InspiratonalQuotesAPI` and open the solution file.

2. **Build and Run:**
   Press `F5` to build and run the solution.

3. **Backend Setup:**
   Once the build is successful, the backend is ready for use.
